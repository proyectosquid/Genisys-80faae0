# Genisys
Feature-rich server software for Minecraft: Pocket Edition & Windows 10 Edition Beta
Genisys is a old fork of PocketMine-MP that existed due to the lack of features and updates on PocketMine-MP.
This is a archive of the version 80faae0 (API 2.0.0 for MCPE 0.15.10) that is not available on the oficial iTXTech/Genisys repository.
